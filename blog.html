1.What is DOM and what is the purpose of DOM?

DOM(Document Object Model) a programming interface that allows us to create, change, or remove elements from the document. We can also add events to these elements to make our page more dynamic.Every HTML element in the DOM tree is a node .DOM functions as a bridge that makes the webpage dynamic using the JavaScript language.One of the main purpose is that we can manipulate css styles ,html tags ,elements etc with javaScript interaction.DOM helps us to create dynamic HTML.

3.What is event bubble?

Event bubbling is a method of event propagation when an event is in an element inside another element, and both elements have registered a handle to that event. In event bubbling, the event is first captured and handled by the innermost element and then propagated to outer elements.In an event, capturing stage starts from top to bottom and when capturing stage reached the target phase, then even bubbling stage starts from bottom to top.we can stop event bubbling by event.stopPropagation(),event.stopImmediatePropagation() etc... when event reached the target phase.

4.What is a callback function and why will you use it?

A callback function is a function that passed as an argument to another function.We use callback functions because Callbacks ensure that the function will not run before the task is completed but will run right after the task is completed. It develops asynchronous JavaScript code and keeps them safe from errors. In JavaScript, the way to create a callback function is to pass it as a parameter to another function and then call it back after the task is completed.
